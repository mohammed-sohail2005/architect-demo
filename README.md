# ArchVision Studio

A premium, luxury-focused architecture firm website built with React.js, Tailwind CSS, and Framer Motion.

## Features

- Sophisticated dark charcoal and gold design palette
- Playfair Display + Inter typography
- Framer Motion scroll-triggered animations
- Filterable portfolio with layout transitions
- Glassmorphism UI effects
- Responsive across all devices

## Tech Stack

- **React.js** (Vite)
- **Tailwind CSS v4**
- **Framer Motion**
- **React Scroll**
- **React Icons**

## Getting Started

```bash
npm install
npm run dev
```
